# SCT_SD_1
A GUI-based Temperature Converter built using Python and CustomTkinter.

The application allows users to enter a temperature in Celsius, Fahrenheit, or Kelvin and instantly displays the equivalent temperature in all three units.

## Features
- Convert from Celsius, Fahrenheit, or Kelvin
- Displays all three temperature units simultaneously
- User-friendly graphical interface
- Input validation for accurate conversions
